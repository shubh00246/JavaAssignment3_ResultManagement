# Student Result Management System – Assignment 3

This project implements a Java-based student result management system.

## Features
- Add students with roll number, name, and 3 subject marks
- Validate marks using custom exception
- Calculate average and display Pass/Fail result
- Handle invalid input, mismatches, and invalid marks
- Demonstrates:
  - try–catch
  - throw & throws
  - custom exception class
  - finally block
  - arrays of objects
